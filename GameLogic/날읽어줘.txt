///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
//
//			프로젝트 설명
//
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

1, ConsoleTester
==> 게임로직 Com DLL 을 테스트하는 코솔 프로그램

2, gameLogic_SUN
==> DB 에 있는 SUN 바이너리 데이트를 XML 파일로 변환(Decode) 또는 그 역으로 변환(Encode) 하는 Com DLL

3, gameLogic_SunPs
==> Com Project 에 기본적으로 생성되는 프로젝트므로 신경쓰지 마시길~

4, XMLParser
==> XML 읽고, 쓰는 라이브러리 (오픈소스)

///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////
//
//			폴더 구조 설명
//
///////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

1, _bin_XX (XX 는 다국어 정보임 cn(중국), en(영어), jp(일본), ko(국내), tw(대만) 
==> 바이너리 정보

2, include_XX
==> 썬 라이브러리 헤더 파일을 국가별로 가지고 있어야 한다.

3, lib_XX
==> 썬 라이브러리 라이브러리 파일을 국가별로 가지고 있어야 한다.

4, WopsConfig
==> 게임로직을 테스트 하기 위해서는 폴더를 C 드라이브 밑으로 복사한다.
    그리고 WopsConfig/SUNGMS/XX 에 국가별로 스크립트 파일들을 복사한다.

% 초기에  빌드 하시려고 하시면 제가(문상현) 가지고 있는 파일들을 드릴테니 가지러 오세요~
