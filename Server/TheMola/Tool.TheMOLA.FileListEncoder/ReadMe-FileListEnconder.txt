TheMola.FileListEncoder_d.exe [enter key]

참조하는 파일 "ScriptList.txt"

현재 참조 탭
  [COLOUMNS]->SOURCE_PATH
  [DATA] -> ...(파일 목록)


[주의사항]
파일 억세스 권한 등이나 TARGET_PATH등에 대한 처리가 안된다.

예외 처리가 많은 부분이 고려되지 않은 테스트 유닛을 바로 툴 화 시킨 상태.

현재 DECODE루틴을 제거했으며, 암호화 이후 원상복구에 대한 비교 검증은 넣을 필요는 있다.
  (현재 문제는 없다. 라이브러리 변경에 관련해서 문제가 있을 시 은파가 신경쓸 예정)


사용방법

1)	"ScriptList.txt"에 정의된 SOURCE_PATH 값에 해당하는 상대 위치에 암호화할
	스크립트 파일을	넣는다.
2)	해당 스크립트 전체 선택 후, 마우스 오른쪽 클릭해서 읽기전용 속성 제거
3)	해당 툴을 실행시킨다.
4)	해당 파일은 암호화된 상태로 변경된다.
